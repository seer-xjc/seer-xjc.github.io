    基础命令

    Linux 命令行系统是基于 Shell 的一种用户与操作系统内核交互的方式，是 Linux 系统操作的核心组成部分。通过键入命令，用户可以对系统进行几乎所有的操作，如文件管理、进程控制、系统配置、网络通信、软件安装等。Linux具有非常完备的命令行系统，各种命令、参数的灵活组合可以完成各种精妙的操作。

    Linux 命令行系统虽然庞大而多样，但对于初学者而言，只需掌握一些常用的基础命令，便足以像使用 Windows 一样高效地操作 Linux 系统。下面文档中记录了Linux系统中一些最基础的常用命令（from： 王道训练营cpp-Linux讲义）。

    更加复杂的高阶命令可在实践过程中边用边学，切忌死记硬背。
